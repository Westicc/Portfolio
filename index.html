PROJECTS

Featured Project

Attendify – Smart Attendance Monitoring System

A project prototype that automates attendance tracking using camera-based seat occupancy detection. The system determines presence based on whether a chair is occupied, eliminating the need for manual roll calls. Developed in collaboration with SVPSPC, this project enhances efficiency and accuracy in attendance monitoring through AI-driven image processing.

Collaborative Projects

Barangay Management System

Our team collaborated with Barangay Nueva Victoria in Mexico, Pampanga to develop a Barangay Management System that streamlines local governance and enhances community engagement. This web-based platform allows residents to request documents online, report concerns, provide feedback, and access information about current barangay officials and its history. Additionally, it features an interactive map for better navigation and accessibility. By integrating these digital solutions, our project aims to improve efficiency, transparency, and communication between barangay officials and residents.
